# AI Chat Game

Web application where users can chat with AI characters.

## Stack

- **React** — frontend UI
- **Vite** — build tool and dev server
- **TypeScript** — type safety

## Features

- Chat with AI-powered characters
- Character selection and profiles
- Real-time messaging interface
- Responsive design

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
